MCP Versions

  • For versions older than 1.5, use RetroMCP (NOTE: RetroMCP v1.0-pre3 is preferred because while it has less versions to choose from, it automatically creates a workspace for you)
  • For versions 1.7.10 - 1.12.2, use MCP
